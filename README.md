Projet Web Semantique
FilmliF - H4203

Filmlif est une application qui génère une fixhe d'un film sous forme HTML.
La DBPédia utilisée est la DBPédia FR.
-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=--=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
Gestion des dépendances: ajouter en "externals JARs" tous les JARs contenus dans le dossier ./jars

Lancer l'application (le main est dans MOM-WS/RDFSearch/src/ihm/IHM.java)
Dans le champs pour input un texte, entrer le nom du film dont vous souhaitez avoir la fiche (la casse est à prender en compte).
Cliquer sur ok.
Votre fiche sera ouverte sur votre navigateur préféré - et vous pouvez la retrouver à tout moment dans le dossier du projet (MOM-WS/RDFSearch/).

Amusez-vous bien hohoho!
